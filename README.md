# 🎓 Student Profile Website

A simple web-based **Student Profile Website** developed using **HTML, CSS, and JavaScript** to demonstrate the use of **Git branching, merging, and version control**.

This project includes multiple sections such as Login, Dashboard, Profile, Payment, and Notifications, each developed as individual features and later merged into the main branch.

---

## 📌 Project Objective

The objective of this project is to:

- Build a basic student profile management webpage.
- Practice Git branching and merging.
- Understand collaborative development using GitHub.
- Implement basic HTML, CSS, and JavaScript concepts.

---

## ✨ Features

### 🔐 Login
- Username and password fields
- Login validation using JavaScript
- Displays success or error messages

### 📊 Dashboard
- Welcome message
- Displays available courses
- Dynamic course list generation

### 💳 Payment
- Fee payment section
- "Pay Now" button

### 👤 Student Profile
- Displays:
  - Name
  - Email
  - Course
  - Academic Year
  - Roll Number
- Skills section
- Edit Profile functionality using JavaScript

### 🔔 Notifications
- Exam reminders
- Fee payment reminder
- Course updates
- Library alerts
- Workshop notifications
- Show/Hide notification feature
